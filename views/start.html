Changes from previous prototype
  • Short photo guide v1.1

    USER NEED: As a user I need to know how much of my body needs to be in the photo, so that I don't accidentaly get a photo framed too close to my face

    USER NEED: As a user I need the language around measurements to be consistent between copy and graphic. Action: Content changed on distance graphic to show 0.5 m as 50cm

    STORY: link to jira story

  • Static photo rules page

    USER NEED:

    As a user, if I visit this page I would find the navigation at the end of the page confusing. Action: Removed redundant links at bottom of page

    STORY: PEXD-3140

  • Improve choose your photo method

    USER NEED:As a user who already has a digital photo, I need to clearly understand the options available to me on the choose your photo method page, So that I can go directly to upload my photo

    STORY:PEXRD-17

  • Improve showing prices on the summary page

    USER NEED:As a user, I need to clearly see the price(s) for my passport selections, So that I understand what I'm paying for

    STORY:PEXRD-18

  • Add photo shop guidance to prototype

    USER NEED:As a lab participant, I need the photo shop guide to be added to the prototype, So that it appears when I select the link

    STORY:PEXRD-20
