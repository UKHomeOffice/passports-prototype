Before you continue

It should take around 3 weeks to renew, replace or change a passport. If you're applying for a first UK passport, it should take around 6 weeks but it can take longer.

Use a different service if you need a passport urgently.

Applying for someone else

If you're applying for a child passport or for someone unable to make their own decisions, you need to use a different service.

Significant changes

If your face has changed because of an accident or an illness such as a stroke, you need to apply in a different way.
